## Repository Structure

- `docs/uml/`
- Exported UML diagram images (e.g., `catan-uml.png`)
- `docs/papyrus-model/`
- Papyrus model files (`.uml`, `.di`, `.notation`)
- `docs/reflections/`
- Written reflections for Tasks 2–5 (optional, but recommended)
- `src/papyrus-generated/`
- **Raw** Java code generated by Papyrus (Task 2 evidence).
- This folder is kept unchanged after generation.
- `src/ai-generated/`
- Code generated using a GenAI tool (Task 3 evidence), if applicable.
- `src/main/`
- **Final runnable implementation** (Task 4).
- This is where we copy the generated skeleton and implement the simulation logic.
