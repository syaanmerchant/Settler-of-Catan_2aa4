## Where to Find Everything

### UML + Papyrus Model 
- **Papyrus model files** (`.uml`, `.di`, `.notation`): `docs/papyrus-model/`
- **Exported UML image** (e.g., `.png`): `docs/uml/`

### Papyrus-Generated Code
- **Raw code generated by Papyrus (kept unchanged):** `src/papyrus-generated/`

### GenAI Prompt + GenAI Code 
- **Prompt + tool/model info:** `src/ai-generated/`
- **GenAI-generated code output:** `src/ai-generated/`

### Runnable Implementation
- **Main implementation:** `src/main/CatanAssignment1/`
- **Entry point to run:** `Demonstrator.java`
